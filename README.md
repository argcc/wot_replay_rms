#### Это прога для подсчета среднего квадратического разброса выстрелов в реплеях игры World of Tanks.
**Требования к реплею**
* Не арта
* Включен серверный прицел
* Версия клиента >= 1.0.0

**Как пользоваться**
1. Запустить исполняемый файл из папки target
2. Указать папку с реплеями, пример: E:\Games\World_of_Tanks_RU\replays
3. По желанию поставить галочку "Дополнительная информация" - это опция будет выводить в файл **details.csv** подсчитанный процент отклонения для каждого выстрела каждого реплея.
4. Нажать "Пек-Пек!" и подождать завершения процесса.

**Результат**
После выполнения программы в указанной папке будут созданы файлы:
**errors.txt** - список возникших ошибок (файл не создается, если ошибок не возникло).
**details.csv** - содержит процент отклонения (центр прицела - 0%, край окружности прицела - 100%) каждого выстрела каждого реплея и время выстрела от начала боя в секундах. Файл открывается программой Excel. Ну или блокнотом. Файл создается только при включенной опции "Дополнительная информация".
**stats.csv** - собранная статистика. 
Имеются следующие столбцы:
* **Shots** - количество выстрелов, по которым вычислен результат других столбцов. Чем меньше это значение, тем больше степень погрешности, так что не следует делать выводы по малому количеству выстрелов.
* **Average displace** - среднее значение раброса выстрелов.
* **Root mean square displace** - [средний квадратический](https://ru.wikipedia.org/wiki/Среднее_квадратическое) разброс выстрелов.

Данные разбиты на группы:
* **By vehicle** - статистика для каждого танка отдельно.
* **By date (displace 5 hours)** - статистика для каждой отдельной даты. Дата и время реплея берется из внутренностей самого реплея(т.е. дата создания файла не влияет). "displace 5 hours" означает, что от даты реплея отнимается 5 часов при вычислении его даты. Это значит, что реплей, сыгранный 5-го числа в 2 часа ночи, будет считаться за реплей на 4-е число.
* **By battle minute** - статистика на каждую минуту боя по отдельности.
* **Total** - статистика по всем выстрелам всех просканированных реплеев.
